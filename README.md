# conffig

A TypeScript library built with [bunup](https://bunup.arshadyaseen.com/).

## Installation

```bash
npm install conffig
```

## Usage

```typescript
import { greet } from 'conffig';

console.log(greet('World')); // Hello, World!
```

## Development

```bash
# Install dependencies
bun install

# Build
bun run build

# Develop with watch mode
bun run dev

# Run tests
bun run test
```
